Name: Declick
Link: https://home.snafu.de/wahlm/dl8hbs/declick-0.6.5.tar.gz
Description: declick is a dynamic digital declicker for audio sample files.
Installation Steps: 
1) Unpack the .tgz archive.
2) If you are not using Linux, edit the Makefile
3) Type "make all"
4) Type "make install" (this installs declick in /usr/local/bin )
